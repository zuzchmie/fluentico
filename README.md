# Fluentico - idea powstania
Fluentico to prosta aplikacja webowa do nauki języka angielskiego przy użyciu fiszek, quizów i testów.  
Została stworzona z myślą o osobach starszych, dlatego oferuje czytelny, intuicyjny interfejs i nieskomplikowaną obsługę, dostosowaną do potrzeb dzisiejszych czasów.  
Projekt odpowiada na fakt, że coraz więcej seniorów korzysta z komputerów, smartfonów i internetu oraz częściej podróżuje za granicę.  
Fluentico pomaga im w nauce podstawowego słownictwa i komunikacji w języku angielskim, aby czuli się pewniej podczas podróży i w codziennych sytuacjach.

## Funkcje
- fiszki (ulubione, własne fiszki, gotowe zestawy)  
- quiz dnia  
- testy  
- tryb głosowy  
- możliwość dodawania własnych fiszek  


## Jak uruchomić projekt

### Sposób 1 — lokalnie
1. Sklonuj repozytorium:
git clone https://github.com/zuzchmie/fluentico.git

2. Wejdź do folderu projektu:
cd fluentico

3. Otwórz plik startowy w przeglądarce:
website.html

Możesz:
- kliknąć dwukrotnie plik website.html  
- albo przeciągnąć go do okna przeglądarki  


### Sposób 2 — przez link
Bezpośrednio w przeglądarce wpisać:
https://zuzchmie.github.io/fluentico/website.html


## Struktura projektu
- fiszki_main.html – główny widok fiszek  
- grid_ulubione.html – ulubione fiszki  
- twoje_fiszki.html – własne fiszki  
- gotowe_zestawy.html – gotowe zestawy  
- quiz_dnia.html – quiz dnia  
- testy_main.html – testy  
- dodaj_fiszke.html – formularz dodawania fiszek  
- pliki .css – style dla poszczególnych widoków  
- pliki .js – logika quizów, fiszek i testów  


## Jak korzystać z aplikacji
1. Otwórz website.html  
2. Przejdź do zakładki FISZKI, aby:  
   - przeglądać fiszki  
   - dodać własne fiszki  
   - dodać fiszki do ulubionych  
3. Przejdź do zakładki QUIZ DNIA, aby rozwiązać quiz  
4. Przejdź do zakładki TESTY, aby sprawdzić swoją wiedzę
